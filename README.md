# React Feedback App

It allows users to add, delete feedback.

This project goes over all of the fundamentals of React including...

- Components
- JSX
- Props 
- State (Component & App Level)
- Styling
- Handling Events
- Lists & Keys
- Forms


# Usage

### Install dependencies

```bash
npm install
```

### Run

```bash
npm run dev
```

This will run React on port :3000
